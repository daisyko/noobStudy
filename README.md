# noobStudy
This is a self-study repository of Noob Xu

## Prerequisite
* cmake (for windows/linux/unix)
* make (for linux/unix)

## Usage
### Windows
Use the following script to build the project.
```
script/build.bat
```
### Linux
Build:
```
make 
```
Clean:
```
make clean
```

Then you can get the binaries in cmake_build/
